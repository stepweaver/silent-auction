EMAIL COMMUNICATIONS
================================================================================
The system sends the following emails:

1. EMAIL VERIFICATION (Required - cannot opt out)
   - Sent when user first registers
   - Contains verification link (expires in 24 hours)
   - Must be verified before user can bid

2. BID CONFIRMATION (Opt-in only)
   - Disabled by default
   - User must enable in their dashboard
   - Only sent for FIRST bid on each item (not every bid)
   - Contains: item name, bid amount, link to view item

2b. OUTBID ALERT (Opt-in only)
   - Same opt-in as bid confirmation (dashboard checkbox)
   - Sent when user is outbid on an item
   - Throttled: at most 1 per user per item every 30 minutes
   - Contains: item name, current high bid, link to place higher bid

3. WINNER NOTIFICATION (Required - cannot opt out)
   - Sent automatically when auction closes
   - Contains: all items won, total amount due, payment/pickup instructions
   - Includes link to Payment Instructions page

4. SECURITY ALERT (Required - cannot opt out)
   - Sent if alias is accessed from new device/location
   - Helps protect against unauthorized access

5. ADMIN WINNERS LIST (Disabled)
   - Currently disabled to reduce email volume
   - Admins can view winners in the dashboard instead
